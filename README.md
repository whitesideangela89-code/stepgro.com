## How to Deploy

### Option 1: Any Web Host (cPanel, FTP, etc.)
1. Upload ALL files maintaining the folder structure
2. Ensure index.html is in the root
3. Keep css/ and images/ folders intact
4. Access via your domain

### Option 2: Netlify (Free)
1. Drag and drop the entire `iphone-landing` folder to netlify.com/drop
2. Get instant URL

### Option 3: GitHub Pages (Free)
1. Create a new repository
2. Upload all files
3. Enable GitHub Pages in settings
4. Access via username.github.io/repo-name

### Option 4: Vercel (Free)
1. Sign in to vercel.com
2. Import the folder
3. Deploy instantly
